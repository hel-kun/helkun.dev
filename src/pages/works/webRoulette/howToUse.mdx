# へるーれっと使い方
<br/>

## へるーれっととは
<a href='/works/webRoulette'>これ</a>です。Webルーレットです。
<br/>

## 基本的な使い方
1. 項目を追加&入力
1. 比率を設定
2. 「ルーレットを回す」ボタンを押す！
<br/>

## ちょっと応用的な使い方
外部のjsonファイルから各項目を読み込むことができます。
以下のような形式で書いて、「項目を読み込む」ボタンでjsonを読み込んでください。(一応jsonのテンプレートも配布しておきますので、<a href="https://helkun.dev/roulette/template.json"></a>)
```json title='item.json' frame="terminal"
[
  {
    "label": "hogehoge",
    "rate": 1
  },
  {
    "label": "fugafuga",
    "rate": 1
  },
  {
    "label": "piyopiyo",
    "rate": 2
  }
]
```
この形式に沿わないものをインポートしようとしても弾かれますので注意して下さい。

逆に、「項目を保存」ボタンを押すことで、入力した項目をこの形式のjsonにエクスポートすることもできます。
