## 博麗神社例大祭とは
博麗神社例大祭とは、毎年春と秋に開催される東方Projectオンリーの同人即売会イベントです。
基本的に東京ビックサイトで開催され、コミケほどではないですが毎回たくさんの人がこのイベントに参加します。
2024年5月3日で21回目らしいです。歴史が深い...。

## なんだかんだCD制作は順調でした
そんな感じでCD制作は結構ヤバめに見えますが、実は順調でした。
しばらく曲を書いてなかったおかげかモチベはとても高く、爆速で曲が仕上がっていきました。

曲に関しては、とりあえずトラックのテンプレを事前に用意しました。
ミックスはちょっとでも躓いたらとりあえず後回しにしました。ミックスで悩んでいじりすぎちゃうと逆に壊しちゃうというのがよくあるのでそれだけは避けないといけません。

ジャケットイラストについては依頼したいところでしたがそんなお金と時間がなかったためPhotoShopで自作しました。これが結構大変で1週間以上もかかりました。マジでしんどいね...。
次は素敵なイラストを描いてくれる絵師さんに依頼したいですね。

<h3>上海アリス幻樂団のスペースに長蛇の列が！！！</h3>
<h3>いや、列できるの早すぎだろ〜</h3>

東方最大手サークルの上海アリス幻樂団では、8年ぶりに秘封CDが出ます。私も欲しいので急いで並びます。

<br/>

ZUNさんから直々にCDをもらい、ご満悦の状態で自分のスペースに戻ります。

CD多分10枚くらいしか売れないだろ〜とか思ってたら、意外と買ってくださる人が多くてびっくりしました。
中には翡翠さんのコンピアルバムから私の曲を知って気に入ってくださった方もいて感動しました。本当に創作の励みになります！感謝！

最終的に余ったのは9枚だけ。50枚も用意したのにそんなに余るとは...。本当にありがとうございます。

<br/>

もちろん、私も他のサークルを回ってCDを買い漁りました。ここで、サークル参加の悪い部分が発動してしまいます。

一般参加の場合、私は使いすぎ防止で基本的に10千円前後のお金のみ用意して例大祭に臨んでいました。
しかし、サークル参加では常に千円札が供給されます。その結果、散財が発生するわけです。なぜか20千円も溶けていました。
<h3>サークル参加って恐ろしいね！！！</h3>
まあ、買いたかったCDは全て買えたのでよしとしましょう。


## CDの曲について
ここで今回制作したCDの曲について軽く話していこうかと思います。

とりあえずXFDはこれです。
<YoutubePlayer videoId="I-oH5Dj0EMo" />

#### 1曲目：`export type Music = {genre: touhou-style;};`
一番最初に書いた曲です。これは半年前から頭の中で練ってた曲なのですぐに作れました。
最初は途中にドロップが入ってましたが、東方風から離れすぎたので抹殺しました。なけなしのワブルベースだけ残ってます。

タイトルはTypeScriptのコードになりますが、実はこれエラーはきます。

<br/>

#### 2曲目：不穏な不穏な幻想郷
これ、個人的にかなり気に入ってます。作りながらニマニマしてました。結構バカの進行で進んでますが、同じようなフレーズがずっと続いているのである意味安定している曲です。
いや、マジで全人類聞いてくれということでなんとYouTubeにもあげています。
<YoutubePlayer videoId="BBSrp2wfXy8" />

<br/>

#### 3曲目：小さな小さな賢将 ～ Hel's "Intense and Fast" Arrangement Version.
小さな小さな賢将を私流にアレンジしました。これ、一番ミックスが大変でした。
東方風とは思えないくらいのトラック数があるので本当にミックスに苦戦しましたが、まあ無難なところに落ち着いたと思います。
結構激しい感じだと思います。

<br/>

#### 4曲目：芥川龍之介の河童 ～ Hel's Recent Toutou-Style Arrangement Version.
過去にYouTubeであげたアレンジをすこ〜し拡張した感じです。
これ、個人的に気に入ってたので入れることにしました。
元々は落ち着いた曲を書きたいという願望から制作した曲ですが、そんなに落ち着いてないです。

<br/>

#### 5曲目：Battle Enfusist!!
過去にリア友をテーマに作った曲です。そのため、リア友に関する要素がマシマシですが、リア友に合わないくらいかっこよくなってしまいました。
割とそのままCDに入れてますが、CDに入れる際にプロジェクトファイルが消えていたので復元(耳コピ作業)に苦労しました。
サビのメロディを一新したかったのですが、新たに作ったサビがしっくりこなかったので結局そのままです。

<br/>

#### 6曲目：月時計 ～ ルナ・ダイアル
過去に鬼形獣風に再翻訳したものです。過去一思い通りに作れた曲なのでつい入れてしまいました。
ただ、思い通りに行きすぎてあまり弄りたくないと気持ちもあったのですが、なんだかんだで拡張版を作って収録しています。

<br/>

#### 7曲目：遥か彼方、もう星は見えない ～ Destructed Galaxy.
このCDの裏テーマである「銀河」の要素を意識して作った曲です。
多分このアルバムの中ではかなり東方風になっている方だと思います。

<br/>

#### 8曲目：`<TouhouStyle/>`
実は2つ目に作った曲となっています。1曲目とセットで構想を練っていたのでサクッと作れてしまいました。
ザ、終盤って感じにしているつもりです。

<br/>

#### おまけ：Don't Slacking Off!!
元々は電々。新聞4号に書き下ろした曲で東方風ではないです。小町が仕事をサボっていると四季映姫に見つかてしまい追いかけられて説教されるという一連の流れを想像しながら作ったものです。
ただ、新型コロナに感染して39度の熱を出し流れあ制作した曲なので荒い部分がまだ残ってました。
これが心残りだったのでミックスをやり直したりドロップパートを作り直したりしておまけとして頒布しました。
後々YouTubeで出す予定ですが忙しくて動画編集ができてません。しばらくお待ちください。
