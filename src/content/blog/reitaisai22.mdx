どうもこんにちは、へるくんです。研究室生活も始まって大変です(新規性を探すのが大変)。

<br/>

さて、今回5月5日に開催された第二十二回博麗神社例大祭にてサークル参加をしました。2度目のサークル参加となりました。

今回は、ふりずみさん(<a href="https://x.com/Prism3377" target='_blank'>@prism3377</a>)と一緒にCDを作って参加しました。
やっぱCD作るのは楽しいね！！！
<br/>
合作を行うのは初めてでしたし、CD頒布後にはトラブル対応をやる羽目になったので、ここら辺を書き留めて備忘録とします。誰かの参考になるかも...?

## 時は10月...
10月半ば、ほったらかし温泉にいた私はサークル参加がしたくてうずうずしていました。だって前のサークル参加めっちゃ楽しかったんだもん！！！

ただ、サークル参加をするためのCDを作る時間がありません。当時もさまざまなタスクを抱えていました。
<br/>
しかし、頭のいい私(自称)は思いついたわけです。
<br/>
<h3>2人なら4~5曲作るだけでCDが出せる！！！</h3>
<br/>
そして、目の前には一緒にほったらかし温泉に浸かっているふりずみさんがいました。ふりずみさん社会人となったことでCDを出す時間がおそらくない人でした。

<br/>
私「CD出しませんか！出したいですよね！というわけで合作しませんか！」
<br/>
ふりずみ「お、いいですね」
<br/>
私「うおお！やるぞ！」
<br/>
私&ふりずみ「うおおおおおお！！！」
<br/>

とまあ、大体こんな感じの流れで2人でCDを出してサークル参加してしまおうという感じになりました。

## まずはサークル参加申し込み
12月くらいに例大祭サークル参加申込が始まりました。
サークル参加申込は学割がある私が行いました。
<br/>

サークルカット(サクカ)はふりずみさん作です。
<ImgCard title="サークルカット" url="https://res.cloudinary.com/helkun/image/upload/v1747117408/helkun.dev/blog/reitaisai22/circle_cut.png" alt="サークルカット" />

サクカのこのテトリミノはなに？っていう話ですが深い意味はありません。サクカ制作が申込締切ギリギリになったが故、「どうしてこんな目に！に！に！」と2人で叫んでいた結果、蓮メリの手にテトリミノが置けそうだったのでテトリミノが生えました。

この話がよく分からない方は、以下の動画を見よ。

<YoutubePlayer videoId="Soy4jGPHr3g"/>

サークル参加やCD情報の宣伝は、私のTwitter（現X）アカウントで行うことになりました。これは、サークル参加申し込みを私が担当した際に、登録した私のTwitterアカウントがそのまま例大祭公式サイトのサークル参加者一覧に掲載されるためです。
<br/>

また、サークル参加を申し込む段階で、CDの大まかな構想を2人である程度話し合いました。合作曲を作るという構想はこの時にはもうあったわけです。
